## Comandos para executar de forma manual

Inicializa o servidor:

```bash
  python server.py
```
Inicializa o cliente:

```bash
  python client.py
```

## Comandos para executar teste automatizado

Inicia a execução do teste:

```bash
  python driver.py
```
